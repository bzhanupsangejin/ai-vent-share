二、版本说明
================================================================================

【EXE版本（推荐个人用户）】
- 文件名：GitHubDesktopSetup-x64.exe
- 大小：约170-180 MB
- 特点：自动安装，无需管理员权限
- 安装位置：C:\Users\用户名\AppData\Local\GitHubDesktop

【MSI版本（推荐企业用户）】

- 大小：约170-180 MB
- 特点：支持批量部署，可自定义安装位置
- 需要：管理员权限

三、下载方式
================================================================================

【方式1：直接点击链接下载（推荐）】
--------------------------------------
1. 复制以下链接到浏览器：
   https://desktop.github.com/

2. 点击页面上的 "Download for Windows" 按钮

3. 等待下载完成

【方式2：使用命令行下载】
--------------------------------------
打开PowerShell，运行以下命令：

Invoke-WebRequest -Uri "https://central.github.com/deployments/desktop/desktop/latest/win32" -OutFile "$env:USERPROFILE\Desktop\GitHubDesktopSetup.exe"

【方式3：使用浏览器直接下载】
--------------------------------------
复制以下链接到浏览器地址栏：

https://github.com/desktop/desktop/releases/latest/download/GitHubDesktopSetup-x64.exe

浏览器会自动开始下载

八、常见问题
================================================================================

【问题1】下载速度慢怎么办？
--------------------------------------
解决方案：
1. 先配置hosts文件（使用桌面上的"修改GitHub_Hosts.bat"）
2. 刷新DNS缓存
3. 重新下载

【问题2】下载失败怎么办？
--------------------------------------
解决方案：
1. 检查网络连接
2. 尝试使用备用下载地址
3. 使用下载工具（如IDM、迅雷）
4. 从GitHub Releases页面手动下载

【问题3】安装时提示需要管理员权限
--------------------------------------
解决方案：
1. 右键点击安装文件
2. 选择"以管理员身份运行"
3. 或者下载EXE版本（无需管理员权限）

【问题4】安装后无法启动
--------------------------------------
解决方案：
1. 检查系统版本是否为Windows 10或更高
2. 检查是否安装了.NET Framework
3. 重新下载并安装
4. 查看错误日志
