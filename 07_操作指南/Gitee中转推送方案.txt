使用Gitee中转推送到GitHub（详细步骤）
================================================================================

生成时间：2026年02月06日 21:20
问题：无法连接到GitHub
解决方案：Gitee → GitHub 自动同步

================================================================================
第一步：在Gitee创建仓库并导入GitHub仓库
================================================================================

1. 访问Gitee
   网址：https://gitee.com/
   
2. 注册/登录Gitee账号
   - 如果没有账号，点击"注册"
   - 如果有账号，点击"登录"

3. 创建新仓库
   - 点击右上角"+"号
   - 选择"新建仓库"或"从GitHub/GitLab导入仓库"

4. 选择"导入已有仓库"
   - 仓库名称：ai-vent-share
   - 路径：ai-vent-share
   - 导入地址：https://github.com/bzhanupsangejin/ai-vent-share.git
   - 是否开源：选择"私有"或"公开"（建议私有）
   - 点击"创建"

5. 等待导入完成
   - Gitee会自动从GitHub拉取代码
   - 这个过程可能需要1-2分钟

================================================================================
第二步：推送本地代码到Gitee
================================================================================

在PowerShell中执行以下命令：

# 1. 进入项目目录
cd "C:\Users\HYX\Desktop\AI网站"

# 2. 添加Gitee远程仓库（替换为你的Gitee用户名）
git remote add gitee https://gitee.com/你的Gitee用户名/ai-vent-share.git

# 3. 推送到Gitee
git push -f gitee main

# 如果提示需要输入用户名和密码，输入Gitee的用户名和密码

================================================================================
第三步：在Gitee设置自动同步到GitHub
================================================================================

1. 进入Gitee仓库页面
   https://gitee.com/你的用户名/ai-vent-share

2. 点击"管理"
   在仓库页面右上角

3. 找到"仓库镜像管理"
   在左侧菜单中

4. 点击"添加镜像"

5. 填写镜像信息
   - 镜像方向：推送
   - 目标仓库地址：https://github.com/bzhanupsangejin/ai-vent-share.git
   - 用户名：你的GitHub用户名
   - 密码/Token：你的GitHub Personal Access Token
   - 同步方式：自动同步（推荐）或手动同步

6. 点击"添加"

7. 立即同步
   - 点击"立即同步"按钮
   - Gitee会自动将代码推送到GitHub

================================================================================
第四步：验证同步结果
================================================================================

等待3-5分钟后：

1. 访问GitHub仓库
   https://github.com/bzhanupsangejin/ai-vent-share

2. 检查最新提交
   - 应该看到最新的提交："feat: 完整项目提交 - 311条资源，43个分类，RSS订阅，自动化流程"
   - 提交时间应该是刚才

3. 检查文件数量
   - 应该有167个文件

4. 等待GitHub Pages部署
   - 再等待1-3分钟
   - GitHub Pages会自动部署

5. 验证网站更新
   访问：https://bzhanupsangejin.github.io/ai-vent-share/content_index.json
   检查：total_count 应该是 311

================================================================================
如果Gitee也连接失败
================================================================================

如果连接Gitee也失败，可能是网络问题，尝试：

1. 检查网络连接
   ping gitee.com

2. 检查防火墙设置
   - 确保防火墙没有阻止Git
   - 临时关闭防火墙测试

3. 使用手机热点
   - 切换到手机热点网络
   - 重试推送

4. 等待网络恢复
   - 可能是临时网络问题
   - 等待一段时间后重试

================================================================================
替代方案：手动上传
================================================================================

如果所有网络方案都失败，可以手动上传：

1. 访问GitHub仓库
   https://github.com/bzhanupsangejin/ai-vent-share

2. 点击"Add file" → "Upload files"

3. 将以下关键文件拖拽上传：
   - content_index.json（最重要）
   - rss.xml
   - static/indexes/目录下的所有文件
   - index.html
   - ai-index.html

4. 填写提交信息
   "feat: 更新资源到311条"

5. 点击"Commit changes"

6. 等待GitHub Pages部署

注意：手动上传比较繁琐，建议优先使用Gitee中转方案
