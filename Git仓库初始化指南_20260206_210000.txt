五、可能遇到的问题及解决方案
================================================================================

【问题1：Authentication failed】
原因：GitHub凭证过期或未配置
解决：
1. 使用GitHub Personal Access Token
2. 在推送时输入用户名和Token（而不是密码）
3. 或者使用GitHub Desktop（自动处理认证）

【问题2：remote origin already exists】
原因：远程仓库已经添加过
解决：
git remote remove origin
git remote add origin https://github.com/bzhanupsangejin/ai-vent-share.git

【问题3：failed to push some refs】
原因：远程仓库有新提交
解决：
git pull origin main --rebase
git push origin main

【问题4：refusing to merge unrelated histories】
原因：本地和远程仓库历史不相关
解决：
git pull origin main --allow-unrelated-histories

【问题5：Permission denied】
原因：没有仓库写入权限
解决：
1. 确认GitHub账号有权限
2. 检查SSH密钥配置
3. 或使用HTTPS方式（需要输入用户名和Token）
