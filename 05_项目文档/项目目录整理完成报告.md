## 🔄 回滚方案

如果整理后发现问题，可以从归档文件夹恢复：

### 恢复单个文件
```bash
# 从归档恢复文件到根目录
Move-Item "归档/临时文件/文件名" "./"
```

### 恢复所有文件
```bash
# 从归档恢复所有文件
Move-Item "归档/*" "./" -Recurse
```

### 删除归档文件夹
```bash
# 确认无问题后，可删除归档文件夹
Remove-Item "归档" -Recurse -Force
```
