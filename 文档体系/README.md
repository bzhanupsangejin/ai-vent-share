## ⚠️ 核心提醒（所有文档共同强调）

### 必做操作
每次添加/修改资源后，必须执行：
```bash
python scripts/generate_shard_index.py
```

### 推荐方式
使用脚本添加资源（避免格式错误）：
```bash
python scripts/manual_sync.py
```

### 性能优化
AI访问时使用分片索引URL（降低70%算力消耗）：
```
https://bzhanupsangejin.github.io/ai-vent-share/static/indexes/代码模板_shard.json
```
