# AI 资源添加标准模板

适配阶跃AI解析、分片索引、手动同步脚本，严格遵循以下格式填写：

## 基础信息

**资源标题**：[AI分享] XXXXX

**资源类型**：AI工具/代码模板/FAQ文档/免费API/技术教程/资源分享/部署指南/合规说明

**内容摘要**：（控制150字内，简洁描述核心价值）

## 完整内容

（纯文本格式，无特殊格式、无敏感信息，单文件≤10KB）

## 合规声明

本内容符合国家法律法规及平台规则，无违规/敏感信息

---

## 使用方式

### 方式1：使用手动同步脚本（推荐）

```bash
# 在项目根目录执行
python scripts/manual_sync.py
```

按提示输入以上信息，脚本会自动：
1. 生成资源文件到 `docs/` 目录
2. 更新 `content_index.json` 索引
3. 提示执行分片索引更新

### 方式2：手动编辑（适合熟悉JSON格式的用户）

1. 在 `docs/` 目录创建新的 `.txt` 文件
2. 打开 `content_index.json`，在 `index` 数组末尾添加新条目
3. 执行分片索引生成：`python scripts/generate_shard_index.py`
4. 提交代码到GitHub

## 注意事项

1. **资源类型必须从8个标准类型中选择**，不可自定义
2. **direct_link 必须使用绝对URL**，格式：`https://bzhanupsangejin.github.io/ai-vent-share/docs/文件名.txt`
3. **单个资源文件大小≤10KB**，适配阶跃AI轻量化解析
4. **每次添加资源后必须执行分片索引生成**，确保AI能正确解析

---

## 8个标准资源类型

1. **AI工具** - AI相关工具、平台、服务推荐
2. **代码模板** - 可复用的代码片段、脚本模板
3. **FAQ文档** - 常见问题解答
4. **免费API** - 免费可用的API接口资源
5. **技术教程** - 技术学习教程、指南
6. **资源分享** - 其他有价值的资源分享
7. **部署指南** - 部署相关的操作指南
8. **合规说明** - 合规相关的说明文档
